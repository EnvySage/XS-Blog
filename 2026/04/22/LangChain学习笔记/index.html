LangChain 学习笔记：从入门到 RAG 应用

学习日期：2026年4月21日
使用模型：通义千问 (Qwen-Max / Qwen3-Max)

一、基础模型调用

1.1 LLM 流式输出

使用 stream() 方法实现流式输出，逐字打印响应内容。

1.2 聊天模型

聊天模型支持 HumanMessage、AIMessage、SystemMessage 三种消息类型。

1.3 文本嵌入模型

二、提示词模板

2.1 基础 PromptTemplate

2.2 FewShot 少样本模板

2.3 ChatPromptTemplate 消息模板

MessagesPlaceholder 用于注入对话历史。

三、Chain 链式调用

3.1 基础管道 | 操作符

使用 | 管道符连接各组件，实现链式调用。

3.2 多链串联

可以在链中加入多个模型、解析器，实现复杂流程。

3.3 RunnableLambda 自定义函数

RunnableLambda 允许在链中插入任意 Python 函数。

四、会话记忆

RunnableWithMessageHistory 实现多会话记忆管理。

五、向量数据库与 RAG

5.1 内存向量存储

5.2 Chroma 持久化存储

5.3 RAG 应用：向量检索 + LLM

六、学习总结

模块 核心概念 应用场景
提示词模板 PromptTemplate / ChatPromptTemplate 规范化输入、结构化输出
Chain | 管道符、RunnableLambda 多步骤处理流程
输出解析器 StrOutputParser / JsonOutputParser 格式化 LLM 输出
会话记忆 RunnableWithMessageHistory 多轮对话、上下文理解
向量数据库 InMemoryVectorStore / Chroma 文档检索、语义搜索
RAG Retriever + Chain 基于知识库的智能问答

七、后续学习方向

  1. LCEL 表达式语言：深入学习 LangChain Expression Language
  2. Agent：构建自主决策的 AI 代理
  3. 工具调用：让 LLM 调用外部 API
  4. 自定义组件：扩展 LangChain 功能
